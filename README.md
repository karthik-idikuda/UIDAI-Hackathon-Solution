# AadhaarInsight360 - UIDAI Hackathon

## Overview
A comprehensive digital identity analytics platform built for the UIDAI Data Hackathon 2026. This solution provides deep insights into Aadhaar enrolment, updates, and authentication patterns across India, enabling data-driven policy making and operational improvements.

## Features
-   **Data Quality Score**: Automated assessment of data integrity and completeness.
-   **Smart Insights**: AI-generated highlights of key trends and anomalies.
-   **Geospatial Analysis**: Interactive maps showing state and district-level metrics.
-   **Policy Mode**: Simulation tools for impact assessment of new policies.
-   **Secure**: Enterprise-grade security for handling sensitive demographic data.

## Technology Stack
-   **Framework**: Streamlit for rapid data dashboarding.
-   **Analysis**: Pandas, NumPy, Scikit-learn.
-   **Visualization**: Plotly Interactive Charts.
-   **Export**: automated PDF/Excel report generation.

## Usage Flow
1.  **Login**: Secure access for authorized officials.
2.  **Dashboard**: View high-level KPIs and national trends.
3.  **Drill-down**: Select specific states or districts for granular analysis.
4.  **Report**: Generate and download executive summaries.

## Quick Start
```bash
# Clone the repository
git clone https://github.com/Nytrynox/UIDAI-Hackathon-Solution.git

# Install dependencies
pip install -r requirements.txt

# Run the dashboard
streamlit run app.py
```
